# CodeEffects Documentation

Official documentation for [CodeEffects](https://codeeffects.com) — an embeddable business rules engine for .NET featuring a visual rule editor, AI-assisted rule authoring, and no-code capabilities for business users.

## About CodeEffects

CodeEffects has been building business rules automation software since 2009. Our embeddable rules engine lets non-developers manage complex business logic — without waiting on engineering for every rule change. With the v6 release, CodeEffects adds AI/LLM-assisted rule authoring, making it faster than ever to describe rules in plain language and have them translated into working logic.

## What's in this repo

- **Getting started** — installation, setup, and your first rule
- **Rule editor guide** — using the visual, no-code rule editor
- **AI-assisted authoring** — generating and refining rules with natural language
- **API reference** — integrating CodeEffects into your .NET application
- **Tutorials** — end-to-end walkthroughs for common scenarios
