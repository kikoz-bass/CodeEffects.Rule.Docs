# How to Obtain Code Effects

Code Effects consists of two major components: the [Rule Editor](/decision-automation/business-rule-editor) and the [Rule Engine](/decision-automation/business-rule-evaluation).

- ## Rule Editor

  The editor is free with no limitations. Follow the [Editor Implementation](/decision-automation/business-rules-editor-implementation) guide to obtain and add it to your project.

- ## Rule Engine

  The engine is available in multiple editions with different costs of ownership. Use our [Price List Request Form](/business-rules-engine-licensing) to request the current price list.

  <b>IMPORTANT:</b> Neither edition includes distribution rights. See [Code Effects Editions](/content/business-rules-engine-editions) if you plan to *distribute* Code Effects with your product.

  - ### Engine Free

    Reference this edition from [NuGet](https://www.nuget.org/packages/CodeEffects.Rule.Engine). It’s free to use with no limitations except a <b>five-second</b> delay on all calls to `Evaluate()` and `Filter()`. Follow the [Engine Implementation](/decision-automation/business-rules-engine-implementation) guide to add it to your project.

  - ### Engine Subscription

    This edition has a minimum <b>6-month</b> commitment and allows you to use the engine for a low monthly fee in a [<b>SINGLE PRODUCT</b>](/content/business-rules-engine-editions/#Definitions) with no limitations.

    To obtain it, go to the [Code Effects Portal](https://portal.codeeffects.com) and click <b>Purchase New License</b>. You can set up automatic <b>monthly</b> payments by credit card and download your Code Effects package. Follow the [Engine Implementation](/decision-automation/business-rules-engine-implementation) guide to add the Engine Subscription edition to your project.

    The Code Effects package includes a license file, `codeeffects.ce`. Save this file on the same machine or VM that runs your evaluation code. It must be renewed every <b>6 months</b> without needing to recompile or redeploy your project. See the *Engine Subscription Edition* section of the [Engine Implementation](/decision-automation/business-rules-engine-implementation) article for details on using the license file. This edition has the same performance and feature set as the Engine Enterprise edition.

  - ### Engine Enterprise

    This edition allows you to use the engine in a [<b>SINGLE PRODUCT</b>](/content/business-rules-engine-editions/#Definitions) with no limitations.

    To obtain it, either use the [Code Effects Portal](https://portal.codeeffects.com) and click <b>Purchase New License</b>, or email a purchase order to [client.billing@codeeffects.com](mailto:client.billing@codeeffects.com). Follow the [Engine Implementation](/decision-automation/business-rules-engine-implementation) guide to add the enterprise edition to your project.

  - ### Engine Source, Editor Source

    These editions allow use in a [<b>SINGLE PRODUCT</b>](/content/business-rules-engine-editions/#Definitions) with no limitations and include full source code of the Engine or the Editor, respectively. See [Code Effects Editions](/content/business-rules-engine-editions) for details.

    To obtain either one, send a purchase order to [client.billing@codeeffects.com](mailto:client.billing@codeeffects.com). Follow the relevant implementation guides: [Engine Implementation](/decision-automation/business-rules-engine-implementation/) and/or [Editor Implementation](/decision-automation/business-rules-editor-implementation). This edition has the same performance and feature set as the Engine Enterprise edition.

  - ### Full Source

    This edition covers <b>YOUR ENTIRE ORGANIZATION</b> for use of the engine with no limitations and includes full source for both the Engine and the Editor.

    To purchase, send a purchase order to [client.billing@codeeffects.com](mailto:client.billing@codeeffects.com). Follow the [Engine Implementation](/decision-automation/business-rules-engine-implementation) and [Editor Implementation](/decision-automation/business-rules-editor-implementation) guides to add them to your project. This edition has the same performance and feature set as the Engine Enterprise edition.
